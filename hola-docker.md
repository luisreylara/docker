# Comandos mediante docker container
```
docker container --help
```
# Contenedores que ya NO se estan ejecutando
```
docker container ls -a
docker ps -a 
```
# Borrar contenedores
```
docker container rm ID
docker container rm ID ID
docker container prune
```
# Imagenes
```
docker image --help
```
# Listar imagen
```
docker image ls
```
# Borrar imagen
```
docker image rm ID
docker image prune
```
